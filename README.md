## About this Course

This course focuses on building practical skills in data visualization and dashboard creation using tools like Excel and IBM Cognos Analytics. It covers both foundational and advanced concepts, emphasizing how to transform raw data into meaningful insights through visual storytelling.

Throughout the course, I learned how to:

Create basic and advanced charts (bar, line, pie, scatter, histograms, treemaps, and maps) using Excel.
Use pivot tables and PivotCharts to analyze and summarize data.
Design and build interactive dashboards in Excel.
Work with business intelligence tools such as IBM Cognos Analytics to create dynamic and shareable dashboards.
Apply data storytelling principles to effectively communicate insights.

The course includes hands-on labs and a final project, where I developed a complete set of visualizations and an interactive dashboard, strengthening my ability to present data in a clear and impactful way.

Key skills: Data Visualization, Data Storytelling, Excel (advanced), Dashboards, Business Intelligence, IBM Cognos Analytics.
